# Painel CMS

Projeto de painel administrativo desenvolvido em Laravel com foco em aprendizado de backend e arquitetura MVC.

## Funcionalidades
- Autenticação de usuários
- CRUD completo
- Painel administrativo
- Controle de acesso

## Tecnologias utilizadas
- Laravel
- PHP
- MySQL
- HTML, CSS, JavaScript
- Tailwind CSS

## Observação

Este projeto é uma versão demonstrativa de um painel CMS,
desenvolvida para fins de estudo e portfólio.

A versão utilizada em ambiente corporativo possui adaptações
e não é disponibilizada publicamente.
